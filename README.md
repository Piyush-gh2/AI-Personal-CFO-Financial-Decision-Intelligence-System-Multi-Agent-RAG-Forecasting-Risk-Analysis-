# 💼 AI Personal CFO: Financial Decision Intelligence System

## 🚀 Overview
AI Personal CFO is an intelligent financial decision system that analyzes user income, expenses, and savings to provide predictions, risk analysis, and personalized financial recommendations.

This project combines:
- 📊 Machine Learning (Financial Forecasting)
- 🔎 Retrieval-Augmented Generation (RAG)
- 🤖 Multi-Agent Decision System
- 📈 Streamlit Dashboard

---

## 💡 Key Features

- 📊 Predict savings using ML (Linear Regression)
- 📉 Risk analysis based on spending patterns
- 🤖 AI-powered financial recommendations
- 🔎 RAG-based financial insights using FAISS
- 📈 Interactive dashboard using Streamlit

---

## 🏗️ Architecture

User Input → ML Model → Risk Analysis → Multi-Agent Decision → RAG Insights → Dashboard
